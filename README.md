# deno-tutorial
deno tutorial

* [github](https://github.com/denoland/deno)

## install 

* Shell (Mac, Linux):
```
curl -fsSL https://deno.land/x/install/install.sh | sh
```
* PowerShell (Windows):
```
iwr https://deno.land/x/install/install.ps1 -useb | iex
```

* Homebrew (Mac):
```
brew install deno
```

* Chocolatey (Windows):
```
choco install deno
```
* Scoop (Windows):
```
scoop install deno
```
